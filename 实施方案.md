## 一、项目概览

### 1.1 目标

为 VulnClaw 开源项目搭建官方网站 **vulnclaw.com**，作为项目的门面和入口，承担品牌展示、快速上手引导、文档入口三大职责。

### 1.2 当前状态

| 维度 | 现状 |
|------|------|
| 域名 | vulnclaw.com（已持有） |
| 网站目录 | 完全空白，无任何 HTML/CSS/JS 文件 |
| 品牌资产 | 仅有 🦞 emoji + 文字口号，无 Logo 图片/Favicon |
| GitHub | https://github.com/Unclecheng-li/VulnClaw |
| 安装方式 | `pip install vulnclaw` |
| 竞品参考 | OpenClaw（openclaw.org）目前仅居中 Logo，内容空 |

### 1.3 设计理念

**"说人话，打漏洞"** 的产品调性决定了网站风格：

- **硬核但不冷漠** — 暗色主题 + 红/橙强调色，安全工具的专业感
- **直白不啰嗦** — 首屏即给出安装命令和 GitHub 链接
- **少即是多** — 单页面搞定，不搞花哨的多页导航
- **速度优先** — 纯静态站，首屏 < 1s 加载

## 二、技术选型

### 2.1 方案对比

| 方案 | 优势 | 劣势 | 推荐度 |
|------|------|------|--------|
| **纯 HTML + CSS + JS** | 零依赖、极致轻量、部署简单 | 手写样式工作量稍大 | ⭐⭐⭐⭐⭐ |
| Next.js | 功能强大、SSR/SSG | 过重，杀鸡用牛刀 | ⭐⭐ |
| Astro | 静态站友好 | 需 Node 构建 | ⭐⭐⭐ |
| Vite + Vue/React | 组件化开发 | 引入构建链 | ⭐⭐⭐ |

### 2.2 推荐方案：纯 HTML + Tailwind CSS CDN

**理由**：

1. **极速交付** — 无需 `npm install`、无需构建，写完即部署
2. **零维护** — 没有 `node_modules`、没有依赖升级、没有 breaking change
3. **性能最优** — 纯静态 HTML，CDN 一推即上线
4. **Tailwind CDN** — 通过 `<script src="https://cdn.tailwindcss.com">` 引入，开发体验好，生产环境再 PurgeCSS
5. **部署灵活** — GitHub Pages / Cloudflare Pages / Vercel / 腾讯云 COS 随便选

### 2.3 技术栈

```
vulnclaw.com/
├── index.html          # 单页面，所有内容
├── assets/
│   ├── logo.svg        # 🦞 品牌 Logo（待设计）
│   ├── favicon.ico     # 站点图标
│   └── og-image.png    # 社交分享预览图（1200×630）
├── CNAME               # GitHub Pages 自定义域名（内容：vulnclaw.com）
└── README.md           # 网站维护说明
```

## 六、部署方案

### 6.1 推荐方案：GitHub Pages

**理由**：免费、与 GitHub 仓库天然集成、自动 HTTPS、自定义域名

**步骤**：

1. 在 vulnclaw.com 目录创建 Git 仓库，推送到 GitHub（建议新建 `vulnclaw.github.io` 或独立仓库）
2. 在仓库 Settings → Pages 中启用 GitHub Pages
3. 添加 `CNAME` 文件，内容为 `vulnclaw.com`
4. 在域名 DNS 添加 CNAME 记录指向 `<username>.github.io`
5. 等待 SSL 证书自动签发

### 6.2 备选方案

| 方案 | 费用 | 优势 | 适合场景 |
|------|------|------|---------|
| Cloudflare Pages | 免费 | 全球 CDN、自动部署 | 访客分布全球 |
| Vercel | 免费 | 自动部署、预览环境 | 需要 Next.js 时 |
| 腾讯云 COS | 按量 | 国内访问快 | 主要面向国内用户 |
| Netlify | 免费 | 表单处理、函数 | 需要后端能力 |

### 6.3 部署流程

```
本地开发（vulnclaw.com/）
  ↓ git push
GitHub 仓库
  ↓ GitHub Actions（可选）
GitHub Pages / Cloudflare Pages
  ↓ CDN 分发
https://vulnclaw.com
```
