核心特性

开箱即用，让每一次渗透都有章可循

自然语言驱动

用日常语言描述渗透意图，AI 自动识别阶段、选择工具、编排测试流程。无需记住繁琐命令。

  • 自动阶段识别与切换
  • 上下文记忆与推理
  • 多轮自主渗透循环

MCP 工具链

11 个安全 MCP 服务开箱即用，覆盖浏览器自动化、JS 逆向、HTTP 抓包、移动端 Hook 等场景。

  • fetch / Burp / Chrome DevTools
  • Frida / ADB / JADX
  • IDA Pro 二进制逆向

20 个渗透 Skill

7 个核心 Skill 覆盖全流程，13 个专项 Skill 深入 Web / 安卓 / 逆向 / CTF / OSINT 场景，含 138 份参考文档。

  • pentest-flow 全流程编排
  • ctf-web / ctf-crypto / ctf-misc
  • osint-recon 四维信息收集
  • waf-bypass / crypto-toolkit
  • python_execute 精确 payload 构造

高级 Agent 能力

持续性渗透测试 + 推理过程可视化 + Python 代码执行，轻松应对复杂安全场景。

  • 持续性渗透（100轮/周期 × 10周期）
  • 推理过程 `think on/off` 控制
  • python_execute 精确 payload 构造
  • 内置 CVE/知识库安全检索
